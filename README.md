# Personality Prediction Chatbot

## Overview

This project is a Python-based chatbot that predicts personality traits based on a user's responses to a series of questions. The chatbot uses a rule-based scoring system to analyze answers and identify dominant personality characteristics.

The purpose of this project was to practice chatbot development, JSON data handling, user interaction, and decision-making logic while creating an engaging conversational experience.

## Features

* Interactive question-and-answer chatbot
* Personality trait prediction
* JSON-based question database
* Trait frequency analysis
* Personalized personality summary
* User-friendly conversational interface

## How It Works

1. The chatbot loads questions and answer options from a JSON file.
2. The user answers a series of personality-related questions.
3. Each answer is mapped to one or more personality traits.
4. The chatbot counts the frequency of each trait.
5. The most common traits are used to generate a personality profile.
6. A final personality analysis is displayed to the user.

## Example Traits

* Independent
* Kind
* Intelligent
* Creative
* Calm
* Adventurous
